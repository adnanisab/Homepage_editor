Homepage Editor - Instructions





Here are the steps to apply "Homepage Editor" feature on any website.


1 Add below JS and CSS statements in homepage's head part. You can also place it in "HTML Head Section" field. Once it's done, just update indexes.

Click to see Head Code




2 Add below JS in "SM Fields (JS file)" field under "Settings - Staff".

Click to see JS Code




3 Add below CSS in "SM Fields (CSS file)" field under "Settings - Staff".

Click to see CSS Code




4 Add below layout code in "Layouts" field under "Settings - Staff". Once it's done, submit one article.

Click to see Layout Code




5 Add Class in below provided Categories. Make sure to place classes in Parent DIV. Here Parent DIV means main DIV which includes all contents(DIVs) inside. If there is no Parent DIV created, then create it first.
