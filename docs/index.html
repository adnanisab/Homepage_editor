Homepage Editor - Version 1

  1. Header files Place below two files in "Head Section" of both layout..
    <link type="text/css" rel="stylesheet" href="{{PutHName}}/ufields.css" />
    <script type="text/javascript" src="{{PutHName}}/ufields.js"></script>
  2. #blbodymain Make sure to add id="blbodymain" in <body> tag for both "Free Layout".
  3. Logo Add Class hpe-logo and Attribute data-logo-width="700" in both "Free Layout".
    Here "700" is the logo width in pixels as per the template.
  4. AdGroup Add Attributes data-adgroup="11" data-ad-width="300" in both "Free Layout".
    Here "11" is AdGroup number and "300" is the width in pixels as per the template.
  5. Weather (on top side) Add weather code in "Sections" icon in "Site Manager".
    Add Class hpe-weather-? in both "Free Layout".
    Here "?" is Section number which was created in "Sections" icon. We can get this number by visiting hostname.js and look for bottom side section functions.
  6. Search Bar Add Class hpe-gsearch in both "Free Layout".
  7. Social Links Add Class hpe-social in both "Free Layout".
  8. Top Links Add Class hpe-top-links in both "Free Layout".
  9. Bottom Links Add Class hpe-bottom-links in both "Free Layout".
  10. Custom Text Place custom text in "Sections" icon in "Site Manager".
    Add Class hpe-custom-text-? in parent div as shown in screenshot.
    Here "?" is the section number. We can get this number by visiting hostname.js and look for bottom side section functions.
    NOTE : When we place this section script in Preview, Highlights, etc. which creates ".js" file, then you just need to add "\" in ending sript tag such as </\ script>
  11. Custom HTML Place custom HTML code in "Sections" icon in "Site Manager".
    Add Class hpe-custom-html-? in parent div as shown in screenshot.
    Here "?" is the section number. We can get this number by visiting hostname.js and look for bottom side section functions.
    NOTE : When we place this section script in Preview, Highlights, etc. which creates ".js" file, then you just need to add "\" in ending sript tag such as </\ script>
  12. Custom Image (Footer Background, Header Background, etc.) Upload Image in "Sections" icon in "Site Manager".
    Add Class hpe-custom-img-? in parent div as shown in screenshot.
    Here "?" is the section number. We can get this number by visiting hostname.js and look for bottom side section functions.
    NOTE : When we place this section script in Preview, Highlights, etc. which creates ".js" file, then you just need to add "\" in ending sript tag such as </\ script>
