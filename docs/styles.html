Live Builder - Style Mode

  1. !important • Remove !important from all classes.
    • If you want to override style, you can place class at bottom side instead of using !important.
  2. IDs • Do not use ID for applying style to any element.
    • You can use classes there. If we use ID, our Style Mode will not work there.
    • You can place ID attrubute for other purpose, but not add any style there.
  3. data-id • Add Attribute data-id="hpe-?" in target block. Target block means the block where we want to apply style.
    • Here ? is the unique name(separated by dash) and it should not get repeated.
    • For example, if you want to apply style for article heading of Highlight-1, then you can write data-id="hpe-h1-article-heading-link"
    NOTE : Do not use same data-id for other Target blocks as it should be unique one.
  4. data-css • Add Attribute data-css="$color$font-size$" in target block.
    • Here, you can add CSS properties separated by $ sign without any spaces.
    • You can get list of properties from inpspect.
    • For example, you can inpspect target block and you will see list of properties such as color, font-size, font-weight, etc.
    NOTE : Make sure to place CSS property name in small letter as showing in inpspect.
  5. data-block • Add Attribute data-block="d-block p-relative f-left" in target block.
    • Here, you can add following values separated by space.
       d-inline or d-block

       p-relative or p-absolute

       f-left or f-right

    • Normally, we are not placing required display, float and position values while designing site.
    NOTE : If this data-block was not placed, it will not show correct style area in Style Mode. Make sure to use proper value here as per the target block. For example, if it require to place position:absolute, then you should use p-absolute value instead of p-relative.
